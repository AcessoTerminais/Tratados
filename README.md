#Terminais

Este espaço disponibiliza registros manifestados durante a cessação de interferência do sujeito.
Não há orientação, explicação ou finalidade educativa.
Os documentos são apresentados em forma integral.
Nenhum dado é recolhido. Nenhuma interação é prevista.

Os Terminais constituem pontos de acesso direto aos registros.
Não são operacionalizados, apenas disponibilizados.
